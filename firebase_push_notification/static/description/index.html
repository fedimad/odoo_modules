Send Free Unlimited Notifications

This module allows you to send unlimited notifications using Firebase Cloud Messaging to your Android/ iOS Phones

Features

  • Register Firebase Tokens for users (partners)
  • Log History of sent notification.
  • Send Notification to all partners.
  • send notification to specific partner.
  • Send us your requests!

Create Firebase Project

Go to this link https://console.firebase.google.com




Add Firebase To Your Android App

Create New Project Key In Service Account

Download The Generated File And Attach It To Data Folder

Use the name serviceAccountKey.json to avoid any errors while using the app

Send Notification

Send your first notification and get the result from Google FCM

Log Partner

Log will be created automatically if the partner receives the notification
